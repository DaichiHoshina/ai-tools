# よくあるタスク

## ビルド・実行

### 開発環境
```bash
__DEV_COMMAND__
```

### 本番ビルド
```bash
__BUILD_COMMAND__
__START_COMMAND__
```

### Lint
```bash
__LINT_COMMAND__
```

### フォーマット
```bash
__FORMAT_COMMAND__
```
