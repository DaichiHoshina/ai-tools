# 個人情報保護ルール（MCP/外部API共通）

## 原則
Claude Codeの会話はAnthropic APIに送信される。外部ツール（MCP等）から取得したデータも「入力」に該当するため、個人情報をコンテキストに含めない。

## 禁止事項
- user_id、IPアドレス、メールアドレス、電話番号等の個人情報をMCPで取得しない
- AWSアカウントID、RDS ARN等のインフラ構成情報も取得しない
- 個別レコードの生データ取得禁止（集計クエリのみ許可）

## 許可される操作
- 集計クエリ（count, GROUP BY, SUM等）
- サービス名・パス名・ステータスコードレベルのメトリクス
- ダッシュボード定義の取得

## 個別ユーザー調査が必要な場合
→ 該当ツールのUI URLを出力し「このURLで確認してください」と案内

## ファイル保存時
- 個人情報は必ず匿名化（User-A, User-B方式）

## 適用対象MCP
- Datadog: search_datadog_logs の extra_fields に PII を含めない
- Slack: ユーザーのDM内容を取得しない
- Notion: 個人情報を含むページ内容をコンテキストに展開しない
